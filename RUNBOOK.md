# Runbook (starter)

## Colab
- Upload the notebook
- Run cells top-to-bottom
- If Streamlit is launched, follow the printed URL (or your tunnel URL)

## Local
- Create venv, install requirements, run Jupyter
- If the notebook writes Streamlit pages, run Streamlit from the repo root

## Versioning
- Tag releases (e.g., v0.1.0) when you complete a sprint milestone
- Keep a short CHANGELOG entry per sprint
