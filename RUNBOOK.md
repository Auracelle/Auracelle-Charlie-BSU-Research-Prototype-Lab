# Runbook

## Streamlit Cloud
1. Push this repo to GitHub.
2. In Streamlit Cloud:
   - Select the repo
   - Set **Main file path** = `app.py`
3. Deploy.

## Notes
- Pages appear automatically from the `pages/` folder.
- Session state is shared across pages using `st.session_state`.
- Replace stub engines/providers in `src/` with your notebook logic over time.
