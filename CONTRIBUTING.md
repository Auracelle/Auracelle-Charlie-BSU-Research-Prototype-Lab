# Contributing

## Scope
This repository is a research notebook + prototype environment.

## Suggested contributions
- Reproducibility improvements (requirements, environment)
- Documentation (runbook, facilitation guides)
- Refactoring notebook cells into modules
- Measurement instrumentation and export reliability

## Process
1. Fork the repo and create a branch.
2. Keep changes focused and well-described.
3. Open a Pull Request with testing notes (Colab + local if possible).
